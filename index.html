Asesoría Docente con IA

Un espacio diseñado para que los docentes de colegios públicos encuentren herramientas, tutoriales, prompts y recursos de inteligencia artificial que faciliten su planeación de clase, atención a la diversidad y trabajo diario.

🤖 Accede a nuestra Gema en Gemini

¿Qué es este sitio?

Somos un equipo de docentes-investigadores de la Universidad Tecnológica de Bolívar que creamos este espacio para reunir, en un solo lugar, las mejores herramientas de inteligencia artificial aplicadas a la labor docente. Aquí encontrarás tutoriales claros, prompts listos para usar y una Gema de Gemini personalizada que te ayudará a planear tus clases con enfoque inclusivo (DUA), crear fichas, canciones, infografías, evaluaciones y PIAR. Todo pensado para docentes que no son expertos en tecnología. 💚

ChatGPT

Gratis
💬 Habla con ChatGPT

Tutorial completo en español: cómo usar ChatGPT para crear planeaciones, actividades, evaluaciones y más. Ideal para docentes principiantes.

📋 Prompt recomendado
"Eres un docente experto en DUA. Diseña una planeación de clase para grado 2°, área Matemáticas, tema 'Sumas con llevadas', duración 2 horas. Incluye: propósito, momentos de clase con tiempos, adaptaciones para estudiantes con NEE, y una rúbrica de evaluación."
Referencia: Andrés Villalba · Guía ChatGPT para Docentes
🧠

Claude (Anthropic)

Gratis
💬 Habla con Claude

Aprende paso a paso cómo usar Claude AI. Excelente para redactar planeaciones detalladas, crear letras de canciones educativas y generar documentos largos con lenguaje natural.

📋 Prompt recomendado
"Eres un compositor de canciones infantiles. Escribe una canción sobre la fotosíntesis para grado 3°, estilo cumbia colombiana, con estribillo pegajoso. Incluye referencias a la región Caribe."
Referencia: Facundo Corengia · Tutorial Claude AI
🌀

Gemini (Google)

Gratis
💬 Habla con Gemini

Conoce Gemini, la IA de Google. Perfecta para crear materiales educativos, integrar con Google Classroom y usar la Gema personalizada que creamos para docentes colombianos.

📋 Prompt recomendado
"Crea una ficha de actividad imprimible para grado 2°, área Ciencias Naturales, tema 'Los animales de la región Caribe'. Versión: BLANCO Y NEGRO (para colorear). Incluye instrucciones, actividad principal y auto-evaluación con caritas."
Referencia: Curso Prompt Engineering · YouTube
🔍

Deepseek

Gratis
💬 Habla con Deepseek

Deepseek es una IA gratuita con excelente razonamiento. Ideal para crear PIAR, documentos normativos y tareas que requieren precisión y estructura lógica.

📋 Prompt recomendado
"Redacta un PIAR completo según Decreto 1421 de 2017 para un estudiante de grado 3° con diagnóstico de TDAH. Fortalezas: creatividad, expresión oral. Barreras: atención sostenida, lectoescritura. Incluye ajustes por área, recursos y acta de acuerdo."
Referencia: Guía Deepseek para principiantes
💻

Microsoft Copilot

Gratis
💬 Habla con Copilot

Copilot viene integrado con Office 365 (Word, Excel, PowerPoint). Muchos colegios públicos ya tienen acceso. Ideal para crear documentos, presentaciones y análisis de datos educativos.

📋 Prompt recomendado
"Crea una presentación de 5 diapositivas sobre el ciclo del agua para grado 2°. Incluye: definición, etapas (con dibujos sugeridos), dato curioso y una actividad práctica para el aula."
Referencia: Microsoft Copilot Tutorial
📓

NotebookLM (Google)

Gratis
💬 Abre NotebookLM

NotebookLM es un cuaderno de investigación con IA. Sube documentos, PDFs, videos y la IA los analiza, resume y genera podcasts educativos automáticamente.

📋 Prompt recomendado
Sube el PDF de los DBA del MEN a NotebookLM y pregúntale: "Resume los DBA de Ciencias Naturales para grado 2° y sugiere 3 actividades prácticas para cada uno, con materiales que se consiguen en cualquier colegio público."
Referencia: Tutorial NotebookLM · YouTube
🔎

Perplexity AI

Gratis
💬 Busca con Perplexity

Perplexity es un buscador inteligente que te da respuestas con fuentes verificadas. Perfecto para preparar clases con información actualizada y confiable.

📋 Prompt recomendado
"¿Cuáles son las mejores estrategias para enseñar fracciones en grado 3° de primaria? Dame 5 actividades concretas con materiales accesibles para colegios públicos colombianos. Incluye fuentes."
Referencia: Tutorial Perplexity AI Español
🎨

Canva + IA

Gratis
🎨 Abre Canva

Canva ahora tiene Magic Studio con inteligencia artificial. Crea infografías, fichas, carteles y presentaciones en segundos. Ideal para diseñar materiales visuales para tu clase.

📋 Prompt recomendado
Usa el prompt del Módulo 4 (Infografías) de la GEM, pégalo en ChatGPT y luego lleva la descripción a Canva. O directamente en Canva: "Infografía educativa sobre el sistema solar para grado 2°, tonos azules, con dibujos grandes y texto mínimo."
Referencia: Tutorial Completo Canva 2024
📊

Gamma AI

Gratis
📊 Abre Gamma

Gamma crea presentaciones impactantes con solo escribir el tema. Ideal para preparar clases visuales, presentaciones para reuniones de padres y materiales para tus estudiantes.

📋 Prompt recomendado
"Crea una presentación sobre 'La Independencia de Colombia' para grado 4°. 8 diapositivas, estilo visual, incluye fechas clave, personajes principales y una línea de tiempo. Tono: educativo, para niños de primaria."
Referencia: Tutorial Gamma AI · YouTube
🎵

Suno AI

Gratis
🎵 Abre Suno AI

Suno AI genera canciones completas con letra y música a partir de texto. Crea canciones educativas sobre cualquier tema para tus clases. ¡A los niños les encanta!

📋 Prompt recomendado
Paso 1: Pide a ChatGPT la letra. Paso 2: Pega en Suno AI. Ejemplo de letra: Canción infantil estilo vallenato sobre las tablas de multiplicar del 2 para grado 2°. Estribillo: "Dos por uno es dos, dos por dos es cuatro, ¡multiplicar es fácil, solo practico un rato!"
Referencia: Tutorial Suno AI · YouTube
